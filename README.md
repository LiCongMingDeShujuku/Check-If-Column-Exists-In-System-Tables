## 中文
如果你需要检查系统表格下是否存在列（在以下示例中，我们将查找支持包含数据库（contained databases）的[containment]列），这里是一些可能有所帮助的SQL逻辑（logic）。

## English
If you have need to check if a column exists under System Tables (In this example we’ll look for the [containment] column which shows support for contained databases) here’s a quick bit of SQL logic that might help.

---
## Logic
```SQL
use master;
set nocount on
 
--	check for contained databases column under sys.databases
--	检查sys.databases下的包含数据库（contained databases）列
if col_length('sys.databases', 'containment') is null
    begin
        select 'Containment has not been found.'
    end
    else
        select 'Containment has indeed been found.'

```
